GPU - Introduction

Quentin Lurkin

Qu'est-ce qu'un GPU

  • Graphics Processing Unit
  • Aide le processeur principal (CPU)
  • Initialement pour le calcul des graphismes
  • De plus en plus pour d'autres choses.

Pourquoi un GPU?

  • Optimisé pour le calcul en paralèlle:
    • Intel Core i9-9980XE: 18 cores
    • NVidia Titan RTX: 4608 shader cores, 288 texture mapping unit, 96 render output unit, 72 ray tracing cores, 576 tensor cores
  • La même procédure sur un grand nombre de données

Comment utiliser le GPU?

  • 3 constructeurs pour les ordinateurs:
    • Intel: GPU Intégré dans les CPU Intel
    • AMD: Cartes Radeon et Radeon PRO
    • NVidia: Cartes GeForce et Quadro
  • Plusieurs API:
    • DirectX: API graphique de Microsoft
    • Metal: API graphique d'Apple
    • OpenGL/Vulkan: API graphique ouverte
    • CUDA: API de calcul de NVidia
    • OpenCL: API de calcul ouverte

Vulkan

  • API Moderne: Successeur d'OpenGL
  • Ouverte et Cross-plateform
  • API extremement explicite
  • Gagne en popularité très rapidement
  • Graphismes et Calculs
