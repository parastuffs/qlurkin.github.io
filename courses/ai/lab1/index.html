Getting Started with pandas

pandas is an extremely usefull library for loading and preparing data in Python. This library is mandatory in all Data Science related project in Python.

To get started with pandas you should:

  • start by learning about the structures used by pandas,
  • and then get an idea of what you can do with it.

Exercices

Do the following exercices in a Jupyter Notebook File (.ipynb). You have to post your file on claco by the end of the lab.

  1. Create a DataFrame with the data of the Pokemon Data.
  2. Create a Series that contains the names of the Pokemon.
  3. Find the Pokemon that have the most HP.
  4. Add two new columns Atk Tot and Def Tot that are respectively the sum of Attack and Sp. Atk and the sum of Defense and Sp. Def.
  5. Create a Dataframe that only contains the Pokemon of Type 1 Fire.
  6. Create a Series with the names of Pokemon that contains "Mega".
  7. Add a column Rank that contains:
    • "Strong" if Attack and Defense values are above their respective mean value.
    • "Weak" if Attack and Defense values are below their respective mean value.
    • "Standard" in other cases.
  8. Create a Series that have all the different Type 1as index and the number of Pokemon of that index as values.
  9. Compute the survival rate for both gender.
  10. With the Titanic Dataset, group passengers in different age groups (10 years) and plot the survival rate of each group.
  11. Based on the previous results, was the saying "Women an Children First" a reality in the Titanic disaster?
