Lab 3 - QuotesApp

Presentation

In this lab, you will create an App that display quotes loaded from the internet. The App will have two screens: a list of quotes and a detail screen for each quote. You must be able to navigate from the list to the detail of a quote and come back. You can load the quotes from:

https://quentin.lurkin.xyz/courses/mobile/lab3/quotes.json
