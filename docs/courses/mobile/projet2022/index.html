Social Network Projet d'application mobile

Énoncé

On vous demande de créer l'app d'un petit réseau social

Chaque utilisateur créer un compte et s'y connecter. Une fois connecté, il peut créer des posts.

On peut chercher un utilisateur pour accéder à son fil de posts. Et on peut choisir de le suivre pour recevoir des notifications à chaque nouveau post.

Un post peut contenir du texte et des photos. Les autres utilisateurs peuvent liker le post.

Modalités

Le projet est a réaliser par binôme

L'application doit être native. Vous pouvez choisir librement les outils de développement.

Vous devez implémenter une partie serveur. Vous pouvez utiliser des services comme Firebase.

Votre code (app et serveur) doit se trouver sur un git accessible par Quentin Lurkin.

Vous devrez défendre ce projet individuellement lors de l'examen oral en session.

Grille d'évaluation

  • Créer un compte et s'y connecter : 2 points
  • Créer des posts : 3 points
  • Les posts peuvent contenir des photos : 2 points
  • Les posts peuvent être liker : 2 points
  • On peut chercher parmi les utilisateurs : 3 points
  • On peut consulter le fil de posts d'un utilisateur : 3 points
  • On peut suivre un utilisateurs : 2 points
  • L'app met en évidence les nouveaux posts des utilisateurs suivis : 3 points
