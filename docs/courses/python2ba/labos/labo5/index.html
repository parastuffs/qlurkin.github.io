IN2L - Informatique appliquée Manipulations de fichiers textes

Objectifs

  • Savoir ouvrir un fichier texte et lire son contenu
  • Savoir créer un fichier texte et y écrire du contenu
  • Savoir importer un dictionnaire Python depuis un document JSON stocké dans un fichier

Application

  1. Définir une fonction computesum(path) qui reçoit un paramètre:

    • path contient le chemin vers un fichier sur l’ordinateur,

    qui est sensé contenir un nombre par ligne, et qui affiche à l’écran la somme de tous ces nombres. Attention, le fichier peut ne pas exister dans lequel cas un message d’erreur est affiché. Le fichier peut également être vide, dans lequel cas la somme est nulle. De plus, des lignes peuvent être erronées (ne contiennent pas un nombre); ces dernières doivent simplement être ignorées. Voici un exemple d’appel avec le contenu du fichier data.txt et le résultat produit:

    
				13
				7.5
				2.1
				DEUX
				4.4
				100
			
    
				computesum('data.txt')
			
    
				La somme est 127.0

  2. Crééz un fichier texte contenant un document JSON, comme l’exemple minimal ci-dessous. Chargez ensuite le fichier et récupérez son contenu comme un dictionnaire Python. Pour chaque pays, ajoutez une entrée nblang qui contient le nombre de langues officielles. Sauvegardez ensuite le dictionnaire mis à jour dans le même fichier.

    
				{
					"Belgium": {
						"capital": "Brussels",
						"languages": ["french", "dutch", "german"]
					},
					"China": {
						"capital": "Beijing",
						"languages": ["mandarin chinese"]
					}
				}
