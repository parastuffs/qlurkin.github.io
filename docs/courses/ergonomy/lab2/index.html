ER4C - Labo 2 Design efficace

Comment créer des designs efficaces?

La Surcharge

Le meilleur moyen d'avoir un bon design est d'éviter de surcharger la page

Alignement

Même des différences d'un pixel sont inconsciement détectées.

Les couleurs

  • Des couleurs complémentaires
  • Des couleurs principalement neutre
  • Une couleur vive pour les choses importantes
  • Couleurs normalisées, consistantes
  • http://www.colourlovers.com/

Polices

  • Peu de polices différentes (2)
  • Limiter l'usage des polices extravagantes
  • Utiliser le gras, l'italique, les majuscules pour différencier le texte.

Espaces blancs

  • Arme ultime contre la surcharge.
  • Normaliser les espaces (rem)

Utilisations des Espaces blancs

Propiétés CSS personalisées (Custom CSS Property)

  • Evite de répéter une même valeur dans toute la feuille de style.
  • Permet de changer et expérimenter facilement.
        
          :root {
            --base-color-h: 14;
            --base-color-s: 100%;
            --base-color-l: 57%;
            --base-color: hsl(var(--base-color-h),
                              var(--base-color-s),
                              var(--base-color-l));
            --base-color-darken-20: hsl(var(--base-color-h),
                                        var(--base-color-s),
                                        calc(var(--base-color-l) - 20%));
            --length-M: 1rem;
            --length-L: calc(var(--length-M) * 2);
            --length-S: calc(var(--length-M) / 2);
          }

          div {
            margin-bottom: 1rem;
          }

          .base {
            background-color: var(--base-color);
            height: var(--length-M);
          }

          .darken {
            background-color: var(--base-color-darken-20);
            height: var(--length-L);
          }
        
      

        <div class="base"></div>
        <div class="darken"></div>

But de la page

Le but de la page doit être facile à trouver

Contenu

Les utilisateurs survolent plus souvent le texte qu'ils ne le lisent

  • Eviter les grands pavés de texte
  • Utiliser les listes à puces
  • Marquer les mots importants (gras, ...)
  • Droit au but
  • Utiliser un language humain / familier

Au final !
